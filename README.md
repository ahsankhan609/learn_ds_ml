# Learn Data Science and Machine Learning with Python.

This repository is a collection of notebooks and scripts for learning Data Science and Machine Learning with Python.

## Contents

- [Notebooks](notebooks): Notebooks for learning Data Science and Machine Learning with Python.
- [Scripts](src): Scripts for learning Data Science and Machine Learning with Python.

## Usage

Open **`learn_ds_ml.code-workspace`** in Cursor or VS Code so the workspace root is this repo. That keeps notebook file links (e.g. `/notebooks/05_pandas/notes.ipynb`) and the Python interpreter (`.venv/`) working correctly.
